$selToPos — выделяет текст. Общий вид специнструкции: $selToPos[A,B,C,D] Параметр A отвечает за начало выделения. Параметры B, C и D – за конец выделения: D – это параметр, заместо которого вы указываете искомый текст, а B и C – это параметры, которые настраивают, как искать этот текст. Заместо D вы указываете символ/набор символов. Этот символ/набор символов используется для поиска позиции, до которой нужно выделять. C и B настраивают параметры поиска. Если заместо C указать 0, поиск будет производиться от начала текста; а если -1, с конца текста. Если заместо B указать 0, символы, указанные заместо D, будут искаться по отдельности, независимо друг от друга; а если -1, будет искаться сразу весь набор символов, указанный заместо D. A определяет, откуда выделять. Если заместо A указать 0 либо 1, выделение будет происходить с начала текста; если 2, выделение будет происходить с начала текста, кроме первого символа; если 3, выделение будет происходить с начала текста, пропуская первые 2 символа и т. д., т. е. с начала текста миноваться будет A−1 символов. $program Последовательно выполняет указанные шаблоны $codes Последовательно выполняет указанные коды.   • Специнструкции $select, $selword и $selline заменяются на выделенный текст. Например, в шаблоне у вас A$selectB. В текстовом поле у вас 123. Если вы выделите 2 и примените шаблон, то в текстовом поле получится 1A2B3. Можно использовать более одного раза в шаблоне, например: 1$select2$select3 + A = 1A2A3.
    Если на момент применения шаблона никакой текст выделен не будет, специнструкция $select будет игнорироваться, $selword будет заменяться на слово, в котором будет находиться курсор на момент применения шаблона, а $selline будет заменяться на абзац, в котором будет находиться курсор на момент применения шаблона.
  • $datetime — вставляет в текущее местонахождение курсора дату и/или время в указанном формате, например, $datetime[dd.MM.yyyy HH:mm].
  • $sellowercase — делает все буквы в выделенном тексте строчными.

    $selupcase — делает все буквы в выделенном тексте прописными.
  • $selinsertword — то, что вы укажете в квадратных скобках после = и этой специнструкции, будет вставлено между каждым символом в выделенном тексте.
    Например, выделив текст 12345 и применив шаблон $selinsertword=[○], вы получите текст 1○2○3○4○5.
  • $selDeleteWord — то, что вы укажете в квадратных скобках после = и этой специнструкции, будет удалено из выделенного текста.
    Например, выделив текст 12345 и применив шаблон $selDeleteWord=[23], вы получите текст 145.
  • $seltranslit — транслитерирует выделенный текст с русского на английский.
  • $selVerseMode — применяет специнструкцию $selupcase к первой букве каждого абзаца в выделенном тексте.
  • $userMenu — позволяет создать своё собственное . Общий вид специнструкции: $userMenu @TITLE:… @COLUMN:… … Основное содержимое специнструкции $userMenu — это кнопки, которые указываются после @TITLE:… и @COLUMN:…. Наличие @TITLE: и @COLUMN: необязательно, и их отсутствие равносильно отсутствию текста после них.

    После @TITLE: вы можете написать название меню.

    После @COLUMN: (задаёт число столбцов кнопок меню) вы можете указать
    • 0/0,/0,0 — равносильно отсутствию текста после @COLUMN:, равносильно отсутствию @COLUMN:. Число столбцов кнопок меню и для вертикальной, и для горизонтальной ориентации телефона будет браться из настроек (jbak2 Keyboard → Внешний вид → В портретном режиме и В ландшафтном режиме в разделе Меню).
    • x/x,/x,0 — x будет числом столбцов кнопок меню как для вертикальной, так и для горизонтальной ориентации телефона.
    • 0,y — y будет числом столбцов кнопок меню при горизонтальной ориентации телефона; число же столбцов кнопок меню для вертикальной ориентации телефона будет браться из настроек.
    • x,y — x будет числом столбцов кнопок меню при вертикальной ориентации телефона, а y будет числом столбцов кнопок меню при горизонтальной ориентации телефона.
  • $selAsInTheSentences — применяет специнструкцию $selupcase к каждой первой букве, находящейся после символов, указанных в jbak2 Keyboard → Автоввод → Правила автосмены в разделе Переход в верхний регистр → Символы смены регистра. Работает, несмотря на пробелы.
  • $paste — альтернатива кода -321.
  • $selReplace — ищет в выделенном тексте указанный текст и заменяет его на другой текст, либо удаляет.

    Для замены пропишите $selReplace @SEARCH:A @REPLACE:B Текст A будет заменён на текст B.

    Для удаления пропишите $selReplace @SEARCH:A @REPLACE:@[DEL] Текст A будет удалён.
