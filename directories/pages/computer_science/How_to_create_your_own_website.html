Шаг 1. Регистрируемся на GitHub Для этого перейдите на главную страницу и нажмите Sign up в левом верхнем углу. Либо же можете воспользоваться прямой ссылкой. Шаг 2 Переходим на страницу создания нового репозитория.

Под словом Owner будет написано ваше имя пользователя, которое вы указали при регистрации. Если в вашем имени пользователя есть большие буквы, делаем их маленькими, после чего к имени пользователя добавляем .github.io и вставляем получившуюся конструкцию в поле Repository name.

Например, если ваше имя пользователя Test, то в поле Repository name вы должны написать test.github.io.

Больше на этой странице ничего не делаем и нажимаем Create repository. Шаг 3 Если созданный репозиторий не открылся у вас автоматически, вы можете найти его на главной странице, либо нажав на . В нашем примере это будет Test/test.github.io.

Итак, мы находимся на странице репозитория. Найдите следующий текст: Get started by creating a new file or uploading an existing file. We recommend every repository include a README, LICENSE, and .gitignore. Нажимаем на LICENSE.

Теперь нам нужно добавить WTFPL license. Переходим на официальную страницу лицензии, копируем текст, возвращаемся на ГитХаб и вставляем скопированный текст в поле Enter file contents here. После этого нажимаем Commit changes... и подтверждаем. Шаг 4 Шаг 5
