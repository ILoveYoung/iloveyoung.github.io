Шаг 1. Регистрируемся на GitHub Для этого перейдите на главную страницу и нажмите Sign up в левом верхнем углу. Либо же можете воспользоваться прямой ссылкой. Шаг 2. Создаём репозиторий После того, как ваш аккаунт будет создан, вы окажетесь на главной странице. Нажмите кнопку Create repository. Либо же можете воспользоваться прямой ссылкой. Под словом Owner будет написано ваше имя пользователя, которое вы указали при регистрации. Если в вашем имени пользователя есть большие буквы, делаем их маленькими, после чего к имени пользователя добавляем .github.io и вставляем получившуюся конструкцию в поле Repository name. Например, если ваше имя пользователя UserNickName111, то в поле Repository name вы должны написать usernickname111.github.io.

Больше на этой странице ничего делать не нужно. Листаем вниз и нажимаем Create repository. Шаг 3. Добавляем WTFPL license После создания репозитория, вы оказываетесь на его главной странице. Листните немного вниз. Нам нужен следующий текст: Нажмите на LICENSE. Вы окажетесь на следующей странице:

Переходим на официальную страницу лицензии, копируем текст, возвращаемся на ГитХаб и вставляем скопированный текст в поле Enter file contents here. После этого нажимаем Commit changes... И подтверждаем. После этого переходим на главную страницу репозитория. Вы всегда сможете найти свои репозитории в разделе Repositories: на главной странице ГитХаба вверху, или в меню внизу (кнопка в левом верхнем углу).
