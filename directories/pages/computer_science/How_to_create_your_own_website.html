Шаг 1. Регистрируемся на GitHub Для этого перейдите на главную страницу и нажмите Sign up в левом верхнем углу. Либо же можете воспользоваться прямой ссылкой. Шаг 2. Создаём репозиторий После того, как ваш аккаунт будет создан, вы окажетесь на главной странице. Нажмите кнопку Create repository. Либо же можете воспользоваться прямой ссылкой. Под словом Owner будет написано ваше имя пользователя, которое вы указали при регистрации. Если в вашем имени пользователя есть большие буквы, делаем их маленькими, после чего к имени пользователя добавляем .github.io и вставляем получившуюся конструкцию в поле Repository name. Например, если ваше имя пользователя TestNickName111, то в поле Repository name вы должны написать следующий текст: testnickname111.github.io Больше на этой странице ничего делать не нужно. Листаем вниз и нажимаем Create repository. Шаг 3. Добавляем WTFPL license После создания репозитория, вы оказываетесь на его главной странице. Листните немного вниз. Нам нужен следующий текст: Нажмите на LICENSE. Вы окажетесь на следующей странице:

Переходим на официальную страницу лицензии, копируем текст, возвращаемся на ГитХаб и вставляем скопированный текст в поле Enter file contents here. После этого нажимаем Commit changes... И подтверждаем. После этого переходим на главную страницу репозитория. Вы всегда сможете найти свои репозитории в разделе Repositories: на главной странице ГитХаба вверху, или в меню внизу (кнопка в левом верхнем углу). В нашем примере главная страница репозитория находится по следующей ссылке: https://github.com/TestNickName111/testnickname111.github.io Шаг 4. Создаём главную страницу сайта Нажимаем на Нажимаем Create new file. В поле Name your file... напишем index.html (это обязательно именно так), а в поле После чего сохраняем файл по уже знакомой схеме (2 раза нажать Commit changes).

Файл можно наполнять содержимым сразу во время создания (поле Enter file contents here), но в примере он оставлен пустым.

Главная страница вашего сайта будет находиться по следующей ссылке: https://testnickname111.github.io Как редактировать файл Нажимаем View code. Выбираем необходимый файл. Нажимаем на Нажимаем Edit in place. После чего редактируем файл и сохраняем.

После редактирования, но перед сохранением, вы можете нажать на кнопку Preview, чтобы просмотреть, какие изменения вы сделали в файле. Это очень удобно для проверки. Как создать папку
  1. Переходим в окно создания нового файла ( → Create new file; см. раздел «Шаг 4»)
  2. Пишем в поле Name your file... название папки
  3. Вводим /
  4. Пишем название файла
  5. Сохраняем
Более правильно это называется не папка (англ. folder), а директория (англ. directory).

Так вот, различие в том, что если в файловом менеджере вы можете создать пустую папку, то вот директория пустой быть не может — в ней должен быть хотя бы 1 файл.
